# 🌱 Smart-Agri-Adviser

An intelligent agricultural decision-support system designed to help farmers optimize crop yields and soil health using data-driven insights.

## 🚀 Features
* **Crop Recommendation:** Recommends the best crop based on soil and weather parameters.
* **Fertilizer Prediction:** Suggests the right fertilizer based on nutrient deficiencies.
* **Disease Detection:** (If applicable) Identifies plant diseases from leaf images.
* **Real-time Weather:** Fetches live weather data to provide timely farming advice.

## 🛠️ Tech Stack
* **Frontend:** [Streamlit](https://streamlit.io/)
* **Language:** Python 3.x
* **Machine Learning:** Scikit-learn, Pandas, NumPy
* **APIs:** OpenWeatherMap (optional)
